Research Project Scope

Literature

The development and use of mobile applications to support children with various learning disabilities, including dysgraphia, dyslexia, dyscalculia, and short-term memory impairments. For dysgraphia, interventions focusing on multisensory techniques and computerized handwriting analysis have shown effectiveness, with applications like Dysgraphicoach providing targeted support. Dyslexia support has seen innovative approaches such as game-based learning, augmented reality, and neurofeedback, emphasizing engagement and continuous assessment. Dyscalculia interventions aim to address early numerical difficulties through personalized, interactive gamified mobile applications. Additionally, mobile apps designed to enhance short-term memory use cognitive training programs and memory games to improve academic outcomes. Overall, these studies underscore the importance of user-friendly, engaging, and evidence-based mobile solutions for children with learning disabilities.

Research Problem

Our application represents a dedicated effort to enhance the learning experience of children grappling with disabilities such as dyscalculia, dyslexia, dysgraphia, and short-term memory loss. By integrating personalized learning strategies and adaptive technologies like AI-driven algorithms that tailor difficulty levels based on individual progress, alongside multisensory instructional methods accommodating various learning styles, we aim to foster a supportive and engaging educational environment. Through a range of interactive exercises designed to address specific challenges these children face—such as phonological awareness training for dyslexia, spatial reasoning tasks for dyscalculia, and fine motor skill exercises for dysgraphia—our application seeks to improve retention, comprehension, and overall academic performance. Continuous refinement based on user feedback and collaboration with educators and specialists ensures that our digital exercises remain effective and responsive to the evolving needs of children with learning disabilities.

Main Objective

  • Create Voice Command support, Text recognition, and Interactive Gamified Mobile application for Children who have learning disabilities to improve their learning experience.

Specific Objectives

  • ML-Based Voice recognition Mobile App component for Dyslexia.

  • ML-based Gamified mobile App component for Dyscalculia.

  • ML-Based Text recognition Mobile App component for Dysgraphia.

  • ML-based Gamified progress prediction mobile app component for Short-Term memory loss.

Solution

The final solution of our research focuses on developing an innovative learning support system for children with learning disabilities such as dyscalculia, dyslexia, dysgraphia, and short-term memory loss. The system integrates mobile applications tailored to each specific learning disability, employing advanced machine learning algorithms and user-friendly interfaces to provide personalized interventions. For dysgraphia, the app utilizes neural network architectures and Support Vector Machine methods to recognize and assess handwritten characters, achieving high accuracy in detecting and assisting with letter formation. The dyslexia app incorporates speech-to-text and text-to-speech functionalities, significantly improving phonological processing and reading skills through interactive exercises. The dyscalculia app features gamified mathematical exercises, promoting engagement and learning progress, while the short-term memory loss application employs predictive models to create customized learning environments, enhancing memory retention and cognitive development. Collectively, these tools offer a comprehensive and effective solution to support the educational needs of children with learning disabilities.
